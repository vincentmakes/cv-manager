# ATS 兼容性

## 什么是 ATS？

**ATS**（Applicant Tracking System，求职者追踪系统）是招聘人员和企业用来管理求职申请的软件。当您将简历上传到招聘门户或公司网站时，ATS 会解析文档以提取结构化数据——您的姓名、职位、公司、日期、技能和教育经历。这些数据随后用于关键词匹配、排名和候选人筛选。

如果 ATS 无法正确解析您的文档，您的申请可能会被丢弃或关键信息可能会丢失——即使您的资质完全匹配。

## 内置 ATS 优化

CV Manager 在公开网站上自动生成对 ATS 友好的输出：

- **Schema.org 标记** — ATS 系统可解析的结构化数据（Person、OrganizationRole、EducationalOccupationalCredential 等）
- **语义化 HTML** — 正确的标题层级、article 元素和列表
- **隐藏 ATS 区块** — 页面中嵌入了简历的纯文本版本，供不处理样式 HTML 的解析器使用
- **简洁的打印输出** — 无视觉干扰，内容层级清晰

无需特殊配置——这些功能始终处于启用状态。

## ATS 文档导出

除了内置的网页优化外，CV Manager 还可以生成**专用的 ATS 友好 PDF**，专为上传到招聘门户和 ATS 系统而设计。

### 如何使用

1. 点击管理工具栏中的 **ATS 文档**
2. 调整**缩放**滑块来控制内容密度（50%–150%）
3. 选择您偏好的**纸张大小**（A4 或 Letter）
4. 在弹窗中预览文档
5. 点击**下载 PDF** 保存文件

### 与打印/PDF 的区别

| 功能 | 打印 / PDF | ATS 文档 |
|------|-----------|----------|
| **目的** | 视觉呈现 | 机器解析 |
| **布局** | 带有颜色、图标、时间线的完整设计 | 干净的结构化文本，最少格式 |
| **内容** | 所有可见部分包括时间线 | 除时间线外的所有部分（时间线与 ATS 无关） |
| **缩放控制** | 浏览器打印对话框 | 内置滑块并带实时预览 |
| **生成方式** | 浏览器打印引擎 | 服务器端（pdfmake） |
| **一致性** | 因浏览器而异 | 各处输出完全一致 |

### ATS 成功技巧

!!! tip "求职申请使用 ATS 文档"
    通过招聘门户申请时，请始终上传 ATS 文档（而非打印/PDF 版本）。结构化布局旨在被自动化系统正确解析。

!!! tip "保持技能部分完整"
    ATS 系统严重依赖关键词匹配。确保您的技能部分包含所有相关的技术、工具和方法论——ATS 导出会将它们作为扁平关键词列表以实现更好的匹配。

!!! tip "给人类读者使用打印/PDF"
    当您直接通过电子邮件将简历发送给招聘经理或带到面试时，使用打印/PDF 版本——它具有完整的视觉设计，包含您的主题颜色和时间线。

!!! tip "调整缩放以控制密度"
    如果您的简历较长，尝试将缩放降低到 70–80% 以在每页容纳更多内容。预览会实时更新，您可以找到最佳平衡点。
