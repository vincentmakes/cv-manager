# Sezioni personalizzate

Le sezioni personalizzate consentono di aggiungere qualsiasi contenuto che non rientra nelle sezioni predefinite.

## Creazione di una sezione personalizzata

1. Aprite **Impostazioni** → scheda **Sezioni personalizzate**
2. Fate clic su **Aggiungi sezione personalizzata**
3. Inserite un nome per la sezione
4. Scegliete un tipo di layout (vedere sotto)
5. Opzionalmente scegliete un'**icona di sezione** (stella, libro, link, globo, cuore, premio, valigetta) — visualizzata accanto all'intestazione
6. Salvate — la nuova sezione appare nel CV

## Tipi di layout

| Layout | Ideale per |
|--------|------------|
| **Griglia a 2 colonne** | Elementi accoppiati come lingue e livello di competenza, strumenti e livello di esperienza |
| **Griglia a 3 colonne** | Elementi compatti come premi, pubblicazioni o credenziali brevi |
| **Elenco verticale** | Elementi sequenziali con link opzionali, come volontariato o iscrizioni |
| **Griglia a schede** | Elementi ricchi con titolo, sottotitolo, descrizione e link — come pezzi di portfolio |
| **Link social** | Link a piattaforme con icone specifiche (LinkedIn, GitHub, Twitter/X, YouTube, Instagram, Dribbble, Behance, Sito web, Email, Telefono o Personalizzato) |
| **Elenco puntato** | Elenchi puntati raggruppati, simili ai punti salienti delle esperienze — ottimo per risultati chiave o competenze |
| **Testo libero** | Blocco di testo semplice con interruzioni di riga preservate — simile alla sezione Informazioni personali, utile per lettere di presentazione, dichiarazioni personali o qualsiasi contenuto in forma libera |

## Opzione nascondi titolo

Per i layout griglia, elenco, schede e elenco puntato, ogni elemento ha una casella di controllo **"Nascondi titolo"**. Quando attivata, l'elemento viene visualizzato senza l'intestazione del titolo — utile quando il contenuto si spiega da solo.

## Gestione degli elementi delle sezioni personalizzate

Fate clic su **Gestisci elementi** su qualsiasi sezione personalizzata per aggiungere, modificare, riordinare o eliminare elementi all'interno di quella sezione.

!!! note "Nota"
    Non esiste un limite massimo per le sezioni personalizzate. Potete crearne quante ne desiderate. Potete anche cambiare il tipo di layout di una sezione personalizzata dopo la creazione — basta modificare la sezione e selezionare un layout diverso.
