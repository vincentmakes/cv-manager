# Modification du contenu

## Profil / En-tête

Cliquez sur l'**icône de modification** dans la section d'en-tête. Vous pouvez définir :

- **Nom** et **Initiales** (affichés dans le cercle d'avatar si aucune photo n'est téléchargée)
- **Titre** (votre rôle/poste principal)
- **Sous-titre** (contexte supplémentaire, par ex. département ou spécialisation)
- **Biographie** (résumé professionnel — les sauts de ligne sont préservés)
- **Lieu**
- **E-mail**, **Téléphone**, **LinkedIn** (affichés sous forme de badges de contact)
- **Langues** (langues parlées, par ex. « Anglais, Français, Allemand » — affiché sous forme de badge avec une icône de globe)

## Photo de profil

Dans la boîte de dialogue de modification du profil, vous pouvez télécharger une photo de profil (JPEG, PNG ou WebP). L'image remplace le cercle d'initiales. Pour la supprimer, utilisez l'option de suppression dans la même boîte de dialogue.

## Expérience professionnelle

Chaque entrée d'expérience comprend :

- **Intitulé de poste** et **Nom de l'entreprise**
- **Date de début** et **Date de fin** (laissez la date de fin vide pour les postes actuels — affiche « Present »)
- **Lieu** et **Code pays** (code à 2 lettres comme `us`, `ch`, `fr` — utilisé pour les icônes de drapeaux sur la chronologie)
- **Points clés** — puces décrivant vos réalisations (une par ligne)

## Logos d'entreprise

Chaque entrée d'expérience peut avoir un **logo d'entreprise**. Les logos apparaissent sur la chronologie sous forme d'images au lieu de simples noms d'entreprise.

### Télécharger un logo

1. Modifiez une entrée d'expérience
2. Dans la section **Logo d'entreprise**, cliquez sur **Choisir une image** pour télécharger un fichier (JPEG, PNG ou WebP, 5 Mo max)
3. Une petite image carrée fonctionne le mieux

### Réutiliser des logos

Cliquez sur **Utiliser un existant** pour ouvrir la grille de sélection de logos, qui affiche tous les logos précédemment téléchargés. Cliquez sur l'un d'eux pour le réutiliser — cela évite de télécharger le même logo plusieurs fois.

### Remplissage automatique

Lorsque vous saisissez un nom d'entreprise qui possède déjà un logo (provenant d'une expérience actuelle ou enregistrée), le logo est automatiquement renseigné. Cela fonctionne entre les jeux de données.

### Propagation globale

Lorsqu'un logo est défini, un bouton bascule apparaît : **« Synchroniser le logo sur toutes les expériences [Entreprise] »**. Lorsqu'il est activé :

- Le logo est appliqué à **toutes** les expériences portant le même nom d'entreprise — y compris celles des jeux de données enregistrés
- Les futures expériences pour la même entreprise hériteront automatiquement du logo
- Le désactiver ne **supprime pas** les logos déjà appliqués — cela arrête uniquement la propagation automatique future

### Supprimer un logo

Cliquez sur le bouton **Supprimer** dans la section logo pour dissocier un logo d'une expérience. Le fichier est conservé sur le disque pour réutilisation. Pour supprimer définitivement un fichier de logo inutilisé, ouvrez le sélecteur de logos et cliquez sur le bouton de suppression sur tout logo qui n'est pas utilisé.

## Certifications

Chaque certification a un **nom**, un **organisme**, une **date de délivrance**, et optionnellement une **date d'expiration** et un **identifiant de certification**.

## Formation

Chaque entrée a un **diplôme/titre**, un **établissement**, des **dates de début/fin**, et une **description** optionnelle.

## Compétences

Les compétences sont organisées en **catégories** (par ex. « Langages de programmation », « Outils et plateformes »). Chaque catégorie contient des étiquettes de compétences individuelles. Les catégories peuvent avoir une **icône** sélectionnée dans une liste prédéterminée : code, serveur, base de données, cloud, paramètres, utilisateurs, mallette, processeur, couches, ou par défaut. Si aucune icône n'est choisie, le système en détecte automatiquement une en fonction du nom de la catégorie (par ex. une catégorie nommée « Programmation » obtient automatiquement l'icône code).

## Projets

Chaque projet a un **titre**, une **description**, des **technologies** (affichées sous forme d'étiquettes) et un **lien externe** optionnel.

## Visibilité des éléments

Chaque élément individuel (expérience, certification, entrée de formation, etc.) dispose d'un **bouton de visibilité**. Les éléments masqués sont exclus de l'impression/PDF et de la vue publique, mais restent dans l'administration pour une utilisation future.

## Réordonnancement par glisser-déposer

Dans la plupart des sections, les éléments peuvent être réordonnés en les faisant glisser. L'ordre de tri est enregistré automatiquement.
