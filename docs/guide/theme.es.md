# Tema y apariencia

## Color del tema

Haga clic en el botón **Theme** en la barra de herramientas para abrir el selector de color. Puede:

- Elegir un color de la **rueda de colores**
- Ajustar el **brillo** con el deslizador
- Introducir un **código hexadecimal** específico
- Elegir entre **8 colores predefinidos** (Blue, Emerald, Purple, Red, Orange, Cyan, Indigo, Pink)

El color del tema afecta a los encabezados, bordes, enlaces, etiquetas y elementos de acento en todo el CV.

## Formato de fecha

Vaya a **Settings → Advanced → Date Format** para elegir cómo se muestran las fechas en su CV:

| Formato | Ejemplo |
|---------|---------|
| MMM YYYY | Jan 2020 |
| MMM YY | Jan 20 |
| MMMM YYYY | January 2020 |
| MM/YYYY | 01/2020 |
| MM.YYYY | 01.2020 |
| MM-YYYY | 01-2020 |
| YYYY-MM | 2020-01 |
| YYYY | 2020 (solo año) |
