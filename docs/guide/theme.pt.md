# Tema e Aparência

## Cor do Tema

Clique no botão **Tema** na barra de ferramentas para abrir o seletor de cores. Você pode:

- Escolher uma cor na **roda de cores**
- Ajustar o **brilho** com o controle deslizante
- Inserir um **código hexadecimal** específico
- Escolher entre **8 cores predefinidas** (Azul, Esmeralda, Roxo, Vermelho, Laranja, Ciano, Índigo, Rosa)

A cor do tema afeta títulos, bordas, links, tags e elementos de destaque em todo o CV.

## Formato de Data

Vá em **Configurações → Avançado → Formato de Data** para escolher como as datas aparecem no seu CV:

| Formato | Exemplo |
|---------|---------|
| MMM YYYY | Jan 2020 |
| MMM YY | Jan 20 |
| MMMM YYYY | Janeiro 2020 |
| MM/YYYY | 01/2020 |
| MM.YYYY | 01.2020 |
| MM-YYYY | 01-2020 |
| YYYY-MM | 2020-01 |
| YYYY | 2020 (apenas ano) |
