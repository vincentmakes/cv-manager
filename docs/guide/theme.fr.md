# Thème et apparence

## Couleur du thème

Cliquez sur le bouton **Thème** dans la barre d'outils pour ouvrir le sélecteur de couleur. Vous pouvez :

- Choisir une couleur dans la **roue chromatique**
- Ajuster la **luminosité** avec le curseur
- Entrer un **code hexadécimal** spécifique
- Choisir parmi **8 couleurs prédéfinies** (Bleu, Émeraude, Violet, Rouge, Orange, Cyan, Indigo, Rose)

La couleur du thème affecte les titres, bordures, liens, étiquettes et éléments d'accentuation dans l'ensemble du CV.

## Format de date

Allez dans **Paramètres → Avancé → Format de date** pour choisir l'affichage des dates sur votre CV :

| Format | Exemple |
|--------|---------|
| MMM YYYY | Jan 2020 |
| MMM YY | Jan 20 |
| MMMM YYYY | January 2020 |
| MM/YYYY | 01/2020 |
| MM.YYYY | 01.2020 |
| MM-YYYY | 01-2020 |
| YYYY-MM | 2020-01 |
| YYYY | 2020 (année uniquement) |
