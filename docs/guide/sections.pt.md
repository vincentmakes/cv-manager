# Seções

## Seções Integradas

O CV Manager vem com 7 seções integradas:

| Seção | Descrição |
|-------|-----------|
| **Sobre** | Seu resumo profissional / bio. Suporta texto com múltiplas linhas. |
| **Linha do Tempo** | Gerada automaticamente a partir das suas experiências profissionais. Exibe uma linha do tempo visual com bandeiras de países. |
| **Experiência** | Histórico profissional com cargo, empresa, datas, localização, bandeira do país e destaques em marcadores. |
| **Certificações** | Certificações profissionais com emissor e datas. |
| **Educação** | Diplomas, instituições, datas e descrições. |
| **Habilidades** | Categorias de habilidades agrupadas com tags individuais. Cada categoria pode ter um ícone personalizado. |
| **Projetos** | Projetos em destaque com descrições, tags de tecnologia e links externos. |

## Reordenando Seções

1. Clique em **Configurações** na barra de ferramentas
2. Vá para a aba **Seções e Títulos**
3. Use os **botões de seta** para mover as seções para cima ou para baixo
4. Clique em **Salvar** — a ordem se aplica às visualizações admin e pública

## Renomeando Títulos de Seções

Na mesma aba **Seções e Títulos**, você pode renomear o título de exibição de qualquer seção. Por exemplo, renomeie "Experiência" para "Histórico Profissional" ou "Habilidades" para "Competências Técnicas". Limpe o nome personalizado para reverter ao padrão.

## Ocultando Seções

Cada seção possui um **botão de visibilidade** (ícone de olho) no canto superior direito. Desativá-lo oculta a seção da impressão/PDF enquanto a mantém visível no admin para edição.
