# Thema & Uiterlijk

## Themakleur

Klik op de knop **Thema** in de werkbalk om de kleurkiezer te openen. U kunt:

- Een kleur kiezen uit het **kleurenwiel**
- De **helderheid** aanpassen met de schuifregelaar
- Een specifieke **hexcode** invoeren
- Kiezen uit **8 vooraf ingestelde kleuren** (Blauw, Smaragd, Paars, Rood, Oranje, Cyaan, Indigo, Roze)

De themakleur is van invloed op koppen, randen, links, tags en accentelementen in het gehele CV.

## Datumnotatie

Ga naar **Instellingen → Geavanceerd → Datumnotatie** om te kiezen hoe datums op uw CV worden weergegeven:

| Notatie | Voorbeeld |
|---------|-----------|
| MMM YYYY | Jan 2020 |
| MMM YY | Jan 20 |
| MMMM YYYY | januari 2020 |
| MM/YYYY | 01/2020 |
| MM.YYYY | 01.2020 |
| MM-YYYY | 01-2020 |
| YYYY-MM | 2020-01 |
| YYYY | 2020 (alleen jaar) |
