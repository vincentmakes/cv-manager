# Design & Erscheinungsbild

## Designfarbe

Klicken Sie auf die Schaltfläche **Design** in der Werkzeugleiste, um den Farbwähler zu öffnen. Sie können:

- Eine Farbe vom **Farbrad** auswählen
- Die **Helligkeit** mit dem Schieberegler anpassen
- Einen bestimmten **Hex-Code** eingeben
- Aus **8 Voreinstellungsfarben** wählen (Blau, Smaragd, Lila, Rot, Orange, Cyan, Indigo, Pink)

Die Designfarbe beeinflusst Überschriften, Rahmen, Links, Tags und Akzentelemente im gesamten Lebenslauf.

## Datumsformat

Gehen Sie zu **Einstellungen → Erweitert → Datumsformat**, um zu wählen, wie Daten in Ihrem Lebenslauf angezeigt werden:

| Format | Beispiel |
|--------|----------|
| MMM YYYY | Jan 2020 |
| MMM YY | Jan 20 |
| MMMM YYYY | Januar 2020 |
| MM/YYYY | 01/2020 |
| MM.YYYY | 01.2020 |
| MM-YYYY | 01-2020 |
| YYYY-MM | 2020-01 |
| YYYY | 2020 (nur Jahr) |
