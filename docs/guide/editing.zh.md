# 编辑内容

## 个人资料 / 头部信息

点击头部区域的**编辑图标**。您可以设置：

- **姓名**和**缩写**（如果未上传照片，缩写显示在头像圆圈中）
- **职位**（您的主要角色/岗位）
- **副标题**（附加信息，例如部门或专业方向）
- **简介**（职业摘要 — 换行符会被保留）
- **所在地**
- **邮箱**、**电话**、**LinkedIn**（显示为联系方式徽章）
- **语言能力**（口语能力，例如 "英语、法语、德语" — 显示为带有地球图标的徽章）

## 个人头像

在个人资料编辑对话框中，您可以上传个人头像（JPEG、PNG 或 WebP 格式）。图片将替换缩写圆圈。要删除头像，请在同一对话框中使用删除选项。

## 工作经历

每条工作经历包含：

- **职位名称**和**公司名称**
- **开始日期**和**结束日期**（结束日期留空表示当前在职 — 显示为"至今"）
- **工作地点**和**国家代码**（两位字母代码，如 `us`、`ch`、`fr` — 用于时间线上的国旗图标）
- **要点描述** — 描述您的成就的要点列表（每行一条）

## 公司标志

每条工作经历可以添加**公司标志**。标志会在时间线上以图片形式显示，替代纯文本公司名称。

### 上传标志

1. 编辑一条工作经历
2. 在**公司标志**部分，点击**选择图片**上传文件（JPEG、PNG 或 WebP 格式，最大 5MB）
3. 小尺寸的正方形图片效果最佳

### 复用标志

点击**使用已有标志**打开标志选择网格，其中显示所有之前上传的标志。点击即可复用 — 这样可以避免多次上传同一个标志。

### 自动填充

当您输入的公司名称已有标志（来自当前或已保存的工作经历）时，标志会自动填充。此功能跨数据集生效。

### 全局同步

设置标志后，会出现一个开关：**"将标志同步到所有 [公司名] 的经历"**。启用后：

- 标志将应用于所有同名公司的工作经历 — 包括已保存数据集中的经历
- 未来为同一公司添加的经历将自动继承该标志
- 关闭此开关**不会**删除已应用的标志 — 只是停止未来的自动同步

### 删除标志

点击标志部分的**删除**按钮可取消标志与工作经历的关联。文件会保留在磁盘上以供复用。要永久删除未使用的标志文件，请打开标志选择器并点击未被使用的标志上的删除按钮。

## 资格认证

每项认证包含**名称**、**颁发机构**、**颁发日期**，以及可选的**到期日期**和**证书编号**。

## 教育背景

每条记录包含**学位/头衔**、**院校**、**起止日期**，以及可选的**描述**。

## 技能

技能按**类别**组织（例如"编程语言"、"工具与平台"）。每个类别包含单独的技能标签。类别可以设置从预设列表中选择的**图标**：code、server、database、cloud、settings、users、briefcase、cpu、layers 或 default。如果未选择图标，系统会根据类别名称自动检测（例如，名为"Programming"的类别会自动获得 code 图标）。

## 项目

每个项目包含**标题**、**描述**、**技术标签**（显示为标签），以及可选的**外部链接**。

## 条目可见性

每个单独的条目（工作经历、资格认证、教育记录等）都有一个**可见性开关**。隐藏的条目将从打印/PDF 和公开视图中排除，但会保留在管理端以供将来使用。

## 拖拽排序

在大多数板块中，条目可以通过拖拽来重新排序。排序顺序会自动保存。
