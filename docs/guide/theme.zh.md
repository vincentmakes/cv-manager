# 主题与外观

## 主题颜色

点击工具栏中的**主题**按钮打开颜色选择器。您可以：

- 从**色轮**中选取颜色
- 使用滑块调整**亮度**
- 输入特定的**十六进制色值**
- 从 **8 种预设颜色**中选择（蓝色、翡翠绿、紫色、红色、橙色、青色、靛蓝、粉色）

主题颜色影响整个简历中的标题、边框、链接、标签和强调元素。

## 日期格式

前往**设置 → 高级 → 日期格式**选择简历中日期的显示方式：

| 格式 | 示例 |
|------|------|
| MMM YYYY | Jan 2020 |
| MMM YY | Jan 20 |
| MMMM YYYY | January 2020 |
| MM/YYYY | 01/2020 |
| MM.YYYY | 01.2020 |
| MM-YYYY | 01-2020 |
| YYYY-MM | 2020-01 |
| YYYY | 2020（仅年份） |
