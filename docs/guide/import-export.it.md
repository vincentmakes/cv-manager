# Importazione ed Esportazione

## Esportare il Vostro CV

Cliccate su **Export** nella barra degli strumenti per scaricare il vostro intero CV come file JSON. Questo include tutte le sezioni, gli elementi, le impostazioni e le sezioni personalizzate. Utilizzatelo per i backup o per trasferire il vostro CV su un'altra istanza.

## Importare i Dati

Cliccate su **Import** e selezionate un file JSON precedentemente esportato. Questo sostituisce i dati attuali del vostro CV con quelli importati. Le sezioni personalizzate e tutte le impostazioni sono incluse.

!!! tip
    Esportate il vostro CV prima di importare, in modo da avere un backup dello stato corrente.
