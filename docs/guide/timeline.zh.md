# 时间线

时间线根据您的工作经历自动生成。它显示：

- 公司标志（如果未设置标志则显示公司名称）
- 职位名称
- 日期范围
- 国旗图标（当不同经历之间的国家发生变化时）

## 时间线点击导航

点击时间线上的任何项目，页面会滚动到对应的工作经历卡片并短暂高亮显示。

## 并行岗位（时间线分支）

当两个或更多工作经历在时间上重叠时，时间线可以自动将它们渲染为**并行轨道**：

- 同时进行的职位会分离到**分支轨道**上——主时间线上方的一条升高线
- **S 形曲线连接器**直观地显示分支从主轨道分叉和合并的位置
- 如果并行职位正在进行中（没有结束日期），分支线会延伸到时间线的右边缘
- 分支轨道上的卡片会单独定位，以避免与主轨道卡片重叠

无需任何配置——分支完全根据您的工作经历日期自动生成。如果您希望使用没有分支的平铺时间线，可以在**设置 → 高级 → 时间线：分支**中禁用此功能。

### 何时会创建分支？

当两段经历重叠 **3 个月或更长时间**时，会创建分支。此阈值会过滤掉短暂的过渡期（在离开旧工作前一两个月开始新工作），同时正确显示真正的并行职位。

**短期职位例外：**如果一段经历非常短（例如 2 个月的代理职位），并且完全在另一个职位的时间范围内，它将始终显示在分支轨道上——即使其持续时间不足 3 个月。关键在于其整个时间跨度都包含在另一个职位的日期范围内。

### 分支何时保持连续？

如果您在一系列连续的主要职位旁边保持一个长期职位（例如兼职或副业），分支将保持**连续**——一条与主轨道平行的单一分支线。即使其中一个连续职位太短而无法达到重叠阈值，只要同一个长期职位是共同的锚点，分支就不会合并后再次分叉。

### 何时不会创建分支？

- 两个长期职位之间 **1–2 个月的重叠**被视为工作过渡，而非并行就业。这是最常见的情况——您在旧工作的通知期结束前不久开始新的职位。
- **没有重叠**——没有日期重叠的连续职位保持在主轨道上。
- **水平空间不足**——如果时间线非常紧凑（在较窄的宽度内有许多项目），太窄而无法清晰显示的分支会自动平铺回主轨道。

## 启用/禁用时间线分支

默认情况下，时间线分支功能处于启用状态。您可以在**设置 → 高级 → 时间线：分支**中禁用此功能。禁用后，所有经历将以平铺交替的布局显示在单一轨道上——即使经历在时间上重叠，也不会显示分叉/合并曲线或升高的分支线。

## 时间线日期格式

默认情况下，时间线仅显示年份（例如 "2020 - 2023"）。您可以在**设置 → 高级 → 时间线：仅显示年份**中更改此设置。禁用后，时间线将使用与简历其余部分相同的日期格式。
