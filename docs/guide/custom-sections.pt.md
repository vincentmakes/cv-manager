# Seções Personalizadas

Seções personalizadas permitem que você adicione qualquer conteúdo que não se encaixe nas seções integradas.

## Criando uma Seção Personalizada

1. Abra **Configurações** → aba **Seções Personalizadas**
2. Clique em **Adicionar Seção Personalizada**
3. Digite um nome para a seção
4. Escolha um tipo de layout (veja abaixo)
5. Opcionalmente, escolha um **ícone de seção** (estrela, livro, link, globo, coração, prêmio, maleta) — exibido ao lado do título
6. Salve — a nova seção aparece no seu CV

## Tipos de Layout

| Layout | Melhor Para |
|--------|-------------|
| **Grade de 2 Colunas** | Itens em pares como idiomas e proficiência, ferramentas e nível de experiência |
| **Grade de 3 Colunas** | Itens compactos como prêmios, publicações ou credenciais curtas |
| **Lista Vertical** | Itens sequenciais com links opcionais, como trabalho voluntário ou associações |
| **Grade de Cartões** | Itens ricos com título, subtítulo, descrição e link — como peças de portfólio |
| **Links Sociais** | Links específicos de plataformas com ícones (LinkedIn, GitHub, Twitter/X, YouTube, Instagram, Dribbble, Behance, Website, Email, Telefone ou Personalizado) |
| **Marcadores** | Listas com marcadores agrupados, similar aos destaques de experiência — ótimo para conquistas-chave ou competências |
| **Texto Livre** | Bloco de texto simples com quebras de linha preservadas — similar à seção Sobre, útil para cartas de apresentação, declarações pessoais ou qualquer conteúdo de formato livre |

## Opção Ocultar Título

Para layouts de grade, lista, cartão e marcadores, cada item possui uma caixa de seleção **"Ocultar título"**. Quando ativada, o item é exibido sem o título — útil quando o conteúdo fala por si só.

## Gerenciando Itens de Seções Personalizadas

Clique em **Gerenciar Itens** em qualquer seção personalizada para adicionar, editar, reordenar ou excluir itens dentro dessa seção.

!!! note "Nota"
    Não há limite fixo para seções personalizadas. Crie quantas precisar. Você também pode alterar o tipo de layout de uma seção personalizada após criá-la — basta editar a seção e selecionar um layout diferente.
