# Abschnitte

## Integrierte Abschnitte

CV Manager enthält 7 integrierte Abschnitte:

| Abschnitt | Beschreibung |
|-----------|--------------|
| **Über mich** | Ihre berufliche Zusammenfassung / Bio. Unterstützt mehrzeiligen Text. |
| **Zeitstrahl** | Automatisch aus Ihren Berufserfahrungen generiert. Zeigt einen visuellen Zeitstrahl mit Länderflaggen. |
| **Berufserfahrung** | Berufliche Laufbahn mit Jobtitel, Unternehmen, Daten, Standort, Länderflagge und Aufzählungspunkten. |
| **Zertifizierungen** | Berufliche Zertifizierungen mit Anbieter und Daten. |
| **Ausbildung** | Abschlüsse, Institutionen, Daten und Beschreibungen. |
| **Fähigkeiten** | Gruppierte Fähigkeitskategorien mit einzelnen Fähigkeits-Tags. Jede Kategorie kann ein benutzerdefiniertes Symbol haben. |
| **Projekte** | Ausgewählte Projekte mit Beschreibungen, Technologie-Tags und externen Links. |

## Abschnitte neu anordnen

1. Klicken Sie auf **Einstellungen** in der Werkzeugleiste
2. Wechseln Sie zum Tab **Abschnitte & Überschriften**
3. Verwenden Sie die **Pfeil-Schaltflächen**, um Abschnitte nach oben oder unten zu verschieben
4. Klicken Sie auf **Speichern** — die Reihenfolge gilt für die Admin- und die öffentliche Ansicht

## Abschnittsüberschriften umbenennen

Im selben Tab **Abschnitte & Überschriften** können Sie den Anzeigenamen jedes Abschnitts ändern. Benennen Sie zum Beispiel „Berufserfahrung" in „Karriere" oder „Fähigkeiten" in „Technische Expertise" um. Löschen Sie den benutzerdefinierten Namen, um zum Standard zurückzukehren.

## Abschnitte ausblenden

Jeder Abschnitt hat einen **Sichtbarkeitsschalter** (Augen-Symbol) in der oberen rechten Ecke. Wenn Sie ihn deaktivieren, wird der Abschnitt in der Druck-/PDF-Ausgabe ausgeblendet, bleibt aber im Admin-Bereich zum Bearbeiten sichtbar.
