# Guía del usuario

Esta guía cubre todo lo que puede hacer con la interfaz de administración de CV Manager.

## Temas

- **[Secciones](sections.md)** — Secciones integradas, reordenamiento, renombrado y ocultamiento
- **[Edición de contenido](editing.md)** — Perfil, fotos, experiencia laboral, logotipos, certificaciones, educación, habilidades y proyectos
- **[Secciones personalizadas](custom-sections.md)** — Cree secciones adicionales con 7 tipos de diseño
- **[Línea de tiempo](timeline.md)** — Línea de tiempo generada automáticamente con soporte para empleos paralelos
- **[Tema y apariencia](theme.md)** — Colores y opciones de formato de fecha
- **[Idioma](language.md)** — Cambie el idioma de la interfaz de administración
- **[Conjuntos de datos](datasets.md)** — Guarde múltiples versiones del CV para diferentes audiencias
- **[Impresión y PDF](print-pdf.md)** — Exporte su CV con números de página y división configurables
- **[Importar y exportar](import-export.md)** — Respalde y restaure los datos de su CV

## Actualizaciones y versión

Abra **Configuración** — la versión actual (por ejemplo, `v1.11.0`) se muestra en la esquina inferior izquierda del pie del modal.

CV Manager verifica automáticamente en GitHub si hay versiones más recientes. Si hay una actualización disponible, aparece un banner azul debajo de la barra de herramientas:

> **Actualización disponible: v1.12.0** (está en v1.11.0) — *Novedades*

El enlace "Novedades" abre el registro de cambios. Puede cerrar el banner con el botón X.

!!! note "Nota"
    La verificación de versión se almacena en caché durante 24 horas. Reinicie su servidor (o contenedor Docker) para limpiar la caché y forzar una verificación nueva.
