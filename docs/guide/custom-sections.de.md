# Benutzerdefinierte Abschnitte

Mit benutzerdefinierten Abschnitten können Sie beliebige Inhalte hinzufügen, die nicht in die integrierten Abschnitte passen.

## Einen benutzerdefinierten Abschnitt erstellen

1. Öffnen Sie **Einstellungen** → Tab **Benutzerdefinierte Abschnitte**
2. Klicken Sie auf **Benutzerdefinierten Abschnitt hinzufügen**
3. Geben Sie einen Abschnittsnamen ein
4. Wählen Sie einen Layout-Typ (siehe unten)
5. Wählen Sie optional ein **Abschnittssymbol** (Stern, Buch, Link, Globus, Herz, Auszeichnung, Aktentasche) — wird neben der Überschrift angezeigt
6. Speichern — der neue Abschnitt erscheint in Ihrem Lebenslauf

## Layout-Typen

| Layout | Geeignet für |
|--------|-------------|
| **2-Spalten-Raster** | Gepaarte Einträge wie Sprachen & Kenntnisstufe, Werkzeuge & Erfahrungslevel |
| **3-Spalten-Raster** | Kompakte Einträge wie Auszeichnungen, Publikationen oder kurze Nachweise |
| **Vertikale Liste** | Aufeinanderfolgende Einträge mit optionalen Links, wie ehrenamtliche Arbeit oder Mitgliedschaften |
| **Karten-Raster** | Umfangreiche Einträge mit Titel, Untertitel, Beschreibung und Link — wie Portfolio-Stücke |
| **Social Links** | Plattform-spezifische Links mit Symbolen (LinkedIn, GitHub, Twitter/X, YouTube, Instagram, Dribbble, Behance, Webseite, E-Mail, Telefon oder Benutzerdefiniert) |
| **Aufzählungspunkte** | Gruppierte Aufzählungslisten, ähnlich wie Erfahrungs-Highlights — ideal für zentrale Erfolge oder Kompetenzen |
| **Freitext** | Einfacher Textblock mit beibehaltenen Zeilenumbrüchen — ähnlich dem Über-mich-Abschnitt, nützlich für Anschreiben, persönliche Erklärungen oder beliebige Freitext-Inhalte |
| **Bildraster** | Bildergalerie mit optionalen Beschriftungen — ideal für Portfolios, Zertifikate oder visuelle Arbeitsproben |
| **Zusätzliche Erfahrungen** | Erfahrungseinträge (Jobtitel, Unternehmen, Daten, Highlights, Logo) — erweitert Ihren Werdegang, ohne sich mit dem Hauptabschnitt Berufserfahrung zu vermischen |

## Bildraster

Das **Bildraster**-Layout zeigt Bilder in einem responsiven Raster mit optionalen Beschriftungen an.

1. Erstellen Sie einen benutzerdefinierten Abschnitt und wählen Sie **Bildraster** als Layout-Typ
2. Wählen Sie die Anzahl der **Rasterspalten** (1, 2 oder 3 — Standard ist 3)
3. Klicken Sie auf **Bild hinzufügen**, um Bilder hochzuladen (JPEG, PNG oder WebP, max. 5MB)
4. Fügen Sie optional eine **Beschriftung** unter jedem Bild hinzu

Bilder werden zentriert und behalten ihr Seitenverhältnis bei. Im 3-Spalten-Modus werden Bilder quadratisch angezeigt für ein einheitliches Erscheinungsbild.

## Zusätzliche Erfahrungen

Das Layout **Zusätzliche Erfahrungen** erstellt Erfahrungseinträge, die mit dem integrierten Abschnitt Berufserfahrung identisch sind. Jeder Eintrag umfasst:

- **Jobtitel** und **Unternehmens-/Organisationsname**
- **Start- und Enddaten** (mit optionalem „Aktuell" für derzeitige Positionen)
- **Standort** und **Ländercode**
- **Highlights** (wichtige Erfolge, einer pro Zeile)
- **Optionales Logo** (gleicher Upload-/Wiederverwendungs-Workflow wie Unternehmenslogos)

### Im Karriere-Zeitstrahl anzeigen

Bei der Verwaltung von Einträgen in einem Abschnitt „Zusätzliche Erfahrungen" können Sie **„Einträge im Karriere-Zeitstrahl anzeigen"** aktivieren. Dies fügt die zusätzlichen Erfahrungseinträge in die Hauptzeitstrahl-Visualisierung ein — sie erscheinen neben Ihren regulären Berufserfahrungen mit voller Unterstützung für Verzweigungen, Logos und Daueranzeige.

## Titel-ausblenden-Option

Für Raster-, Listen-, Karten- und Aufzählungs-Layouts hat jedes Element ein Kontrollkästchen **„Titel ausblenden"**. Wenn aktiviert, wird das Element ohne seine Titelüberschrift angezeigt — nützlich, wenn der Inhalt für sich selbst spricht.

## Elemente benutzerdefinierter Abschnitte verwalten

Klicken Sie auf **Elemente verwalten** bei einem beliebigen benutzerdefinierten Abschnitt, um Elemente innerhalb dieses Abschnitts hinzuzufügen, zu bearbeiten, neu anzuordnen oder zu löschen.

!!! note "Hinweis"
    Es gibt keine feste Obergrenze für benutzerdefinierte Abschnitte. Erstellen Sie so viele, wie Sie benötigen. Sie können auch den Layout-Typ eines benutzerdefinierten Abschnitts nachträglich ändern — bearbeiten Sie einfach den Abschnitt und wählen Sie ein anderes Layout.
