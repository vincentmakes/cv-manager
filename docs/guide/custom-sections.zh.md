# 自定义板块

自定义板块让您可以添加不适合内置板块的任何内容。

## 创建自定义板块

1. 打开**设置** → **自定义板块**选项卡
2. 点击**添加自定义板块**
3. 输入板块名称
4. 选择布局类型（见下文）
5. 可选择一个**板块图标**（star、book、link、globe、heart、award、briefcase）— 显示在标题旁边
6. 保存 — 新板块将出现在您的简历中

## 布局类型

| 布局 | 最适用于 |
|------|----------|
| **两列网格** | 成对的项目，如语言和熟练程度、工具和经验水平 |
| **三列网格** | 紧凑的项目，如奖项、出版物或简短的资质证明 |
| **垂直列表** | 带有可选链接的顺序项目，如志愿服务或会员资格 |
| **卡片网格** | 包含标题、副标题、描述和链接的丰富项目 — 如作品集 |
| **社交链接** | 带有平台图标的链接（LinkedIn、GitHub、Twitter/X、YouTube、Instagram、Dribbble、Behance、网站、邮箱、电话或自定义） |
| **要点列表** | 分组的要点列表，类似于工作经历的亮点 — 非常适合展示关键成就或能力 |
| **自由文本** | 保留换行的纯文本块 — 类似于"关于我"板块，适用于求职信、个人陈述或任何自由格式内容 |
| **图片网格** | 带有可选说明文字的图片画廊 — 非常适合作品集、证书或视觉作品展示 |
| **补充经历** | 经历类型条目（职位、公司、日期、亮点、标志） — 扩展您的工作经历，而不与主要工作经历板块混合 |

## 图片网格

**图片网格**布局在响应式网格中显示图片，并支持可选的说明文字。

1. 创建自定义板块并选择**图片网格**作为布局类型
2. 选择**网格列数**（1、2 或 3 — 默认为 3）
3. 点击**添加图片**上传图片（JPEG、PNG 或 WebP 格式，最大 5MB）
4. 可在每张图片下方添加**说明文字**

图片居中显示并保持其宽高比。在 3 列模式下，图片以正方形显示以获得统一的外观。

## 补充经历

**补充经历**布局创建与内置工作经历板块相同的经历条目。每个条目包含：

- **职位名称**和**公司/组织名称**
- **开始和结束日期**（当前职位可选"至今"）
- **工作地点**和**国家代码**
- **要点描述**（关键成就，每行一条）
- **可选标志**（与公司标志相同的上传/复用流程）

### 在职业时间线上显示

在管理补充经历板块的条目时，您可以启用**"在职业时间线上显示条目"**。这会将补充经历条目合并到主时间线可视化中 — 它们会与您的常规工作经历一起显示，完全支持分支、标志和时长显示。

## 隐藏标题选项

对于网格、列表、卡片和要点布局，每个条目都有一个**"隐藏标题"**复选框。启用后，条目将不显示标题 — 适用于内容本身已经足够说明的情况。

## 管理自定义板块条目

点击任何自定义板块上的**管理条目**来添加、编辑、重新排序或删除该板块中的条目。

!!! note "注意"
    自定义板块没有数量限制。您可以根据需要创建任意数量。您也可以在创建后更改自定义板块的布局类型 — 只需编辑板块并选择不同的布局即可。
