# Chronologie

La chronologie est générée automatiquement à partir de vos expériences professionnelles. Elle affiche :

- Les logos d'entreprise (ou les noms d'entreprise si aucun logo n'est défini)
- Les intitulés de poste
- Les périodes
- Les drapeaux de pays (lorsque le pays change entre les expériences)

## Navigation par clic sur la chronologie

Cliquer sur un élément de la chronologie fait défiler la page jusqu'à la carte d'expérience correspondante et la met brièvement en surbrillance.

## Emplois simultanés (embranchement de la chronologie)

Lorsque deux ou plusieurs expériences professionnelles se chevauchent dans le temps d'au moins 1 mois, la chronologie les affiche automatiquement sous forme de **pistes parallèles** :

- Le poste concurrent se sépare sur une **piste secondaire** — une ligne surélevée au-dessus de la chronologie principale
- Des **connecteurs en courbe en S** montrent visuellement où la branche se sépare et rejoint la piste principale
- Si un poste parallèle est en cours (sans date de fin), la ligne de branche s'étend jusqu'au bord droit de la chronologie
- Les cartes des éléments sur la piste secondaire sont positionnées séparément pour éviter le chevauchement avec les cartes de la piste principale

Aucune configuration n'est nécessaire — l'embranchement est entièrement automatique en fonction des dates de vos expériences. Les chevauchements courts de moins d'1 mois (fréquents lors des transitions professionnelles) sont intentionnellement ignorés.

## Format de date de la chronologie

Par défaut, la chronologie affiche uniquement les années (par ex. « 2020 - 2023 »). Vous pouvez modifier cela dans **Paramètres → Avancé → Chronologie : années uniquement**. Lorsque cette option est désactivée, la chronologie utilise le même format de date que le reste de votre CV.
