## Atualizações e Versão

Abra **Configurações** — a versão atual (ex.: `v1.11.0`) é exibida no canto inferior esquerdo do rodapé do modal.

O CV Manager verifica automaticamente no GitHub se há versões mais recentes. Se uma atualização estiver disponível, um banner azul aparece abaixo da barra de ferramentas:

> **Atualização disponível: v1.12.0** (você está na v1.11.0) — *O que há de novo*

O link "O que há de novo" abre o changelog. Você pode dispensar o banner com o botão X.

!!! note "Nota"
    A verificação de versão é armazenada em cache por 24 horas. Reinicie seu servidor (ou container Docker) para limpar o cache e forçar uma nova verificação.
