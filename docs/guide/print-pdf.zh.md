# 打印与 PDF 导出

## 打印简历

点击工具栏中的**打印 / PDF**（或按 `Ctrl+P` / `Cmd+P`）。使用浏览器的"另存为 PDF"选项即可创建 PDF 文件。打印输出会自动排除所有管理控件、隐藏项目和隐藏板块。

## 页码

在**设置 → 打印与导出 → 页码**中启用页码功能。可配置项包括：

- **位置**：页面顶部或底部
- **格式**："Page 1 of 3"、"1 / 3"、"1" 或 "- 1 -"

## 分页控制

两个选项控制内容如何跨页拆分：

- **允许板块分割** — 允许板块从一页开始并在下一页继续（而不是将整个板块推到新页面）
- **允许条目分割** — 允许单个项目（如工作经历卡片）跨页拆分

这两个选项均位于**设置 → 打印与导出**中。如果简历包含较长的板块，启用这些选项可以避免出现过多空白。

## 公开页面打印按钮

通过**设置 → 打印与导出 → 公开打印按钮**切换是否在公开简历页面上显示打印按钮。启用后，访客可以直接打印您的简历。

## PDF 输出最佳实践

!!! tip "缩放"
    大多数浏览器的打印对话框中都有 **缩放** 选项（有时在"更多设置"或"高级"下）。将其设置为 **80–90%** 通常会产生更好的效果——它可以收紧间距，帮助将简历压缩到更少的页面中，同时不影响可读性。

!!! tip "边距"
    在浏览器的打印对话框中将打印边距调整为 **最小** 或 **无**，以获得最大的内容区域。简历本身有内部内边距，因此去除浏览器边距不会导致文字紧贴边缘。

!!! tip "最佳组合"
    将缩放设置为 85% 并配合最小边距，可获得最专业的输出效果。保存前请先预览结果——小幅调整往往能带来显著改善。
