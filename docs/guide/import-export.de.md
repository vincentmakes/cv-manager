# Import & Export

## Ihren Lebenslauf exportieren

Klicken Sie auf **Exportieren** in der Werkzeugleiste, um Ihren gesamten Lebenslauf als JSON-Datei herunterzuladen. Dies umfasst alle Abschnitte, Einträge, Einstellungen und benutzerdefinierten Abschnitte. Verwenden Sie dies für Backups oder um Ihren Lebenslauf auf eine andere Instanz zu übertragen.

## Daten importieren

Klicken Sie auf **Importieren** und wählen Sie eine zuvor exportierte JSON-Datei aus. Dies ersetzt Ihre aktuellen Lebenslaufdaten durch die importierten Daten. Benutzerdefinierte Abschnitte und alle Einstellungen sind enthalten.

!!! tip
    Exportieren Sie Ihren Lebenslauf vor dem Importieren, damit Sie ein Backup des aktuellen Zustands haben.
