# 板块

## 内置板块

CV Manager 提供 7 个内置板块：

| 板块 | 描述 |
|------|------|
| **关于我** | 您的职业摘要/简介。支持多行文本。 |
| **时间线** | 根据您的工作经历自动生成。以可视化时间线形式展示，带有国旗图标。 |
| **工作经历** | 工作履历，包含职位名称、公司、日期、工作地点、国旗和要点描述。 |
| **资格认证** | 专业资格认证，包含颁发机构和日期。 |
| **教育背景** | 学位、院校、日期和描述。 |
| **技能** | 分组的技能类别，包含单独的技能标签。每个类别可以设置自定义图标。 |
| **项目** | 精选项目，包含描述、技术标签和外部链接。 |

## 重新排序板块

1. 点击工具栏中的**设置**
2. 转到**板块与标题**选项卡
3. 使用**箭头按钮**上下移动板块
4. 点击**保存** — 排序同时应用于管理端和公开端

## 重命名板块标题

在同一个**板块与标题**选项卡中，您可以重命名任何板块的显示标题。例如，将"工作经历"重命名为"职业经历"，或将"技能"重命名为"专业技术"。清除自定义名称可恢复为默认名称。

## 隐藏板块

每个板块右上角都有一个**可见性开关**（眼睛图标）。关闭后，该板块将从打印/PDF 输出中隐藏，但在管理端仍然可见以便编辑。
