# User Guide

This guide covers everything you can do with CV Manager's admin interface.

## Topics

- **[Sections](sections.md)** — Built-in sections, reordering, renaming, and hiding
- **[Editing Content](editing.md)** — Profile, pictures, work experience, logos, certifications, education, skills, and projects
- **[Custom Sections](custom-sections.md)** — Create additional sections with 7 layout types
- **[Timeline](timeline.md)** — Auto-generated timeline with parallel job support

- **[Datasets](datasets.md)** — Save multiple CV versions for different audiences
- **[Print & PDF](print-pdf.md)** — Export your CV with configurable page numbers and splitting
- **[Import & Export](import-export.md)** — Backup and restore your CV data

## Updates & Version

Open **Settings** — the current version (e.g., `v1.11.0`) is shown in the bottom-left corner of the modal footer.

CV Manager automatically checks GitHub for newer versions. If an update is available, a blue banner appears below the toolbar:

> **Update available: v1.12.0** (you're on v1.11.0) — *What's new*

The "What's new" link opens the changelog. You can dismiss the banner with the X button.

!!! note
    The version check is cached for 24 hours. Restart your server (or Docker container) to clear the cache and force a fresh check.
