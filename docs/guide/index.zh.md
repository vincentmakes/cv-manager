# 使用指南

本指南涵盖了您在 CV Manager 管理界面中可以进行的所有操作。

## 内容导航

- **[板块](sections.md)** — 内置板块、重新排序、重命名和隐藏
- **[编辑内容](editing.md)** — 个人资料、照片、工作经历、公司标志、资格认证、教育背景、技能和项目
- **[自定义板块](custom-sections.md)** — 使用 7 种布局类型创建额外板块
- **[时间线](timeline.md)** — 自动生成的时间线，支持并行岗位展示
- **[主题与外观](theme.md)** — 颜色和日期格式选项
- **[语言](language.md)** — 切换管理界面语言
- **[数据集](datasets.md)** — 为不同受众保存多个简历版本
- **[打印与 PDF](print-pdf.md)** — 导出简历，可配置页码和分页方式
- **[导入与导出](import-export.md)** — 备份和恢复您的简历数据

## 更新与版本

打开**设置** — 当前版本号（例如 `v1.11.0`）显示在弹窗页脚的左下角。

CV Manager 会自动检查 GitHub 上的新版本。如果有更新可用，工具栏下方会出现蓝色横幅：

> **有可用更新：v1.12.0**（您当前使用 v1.11.0）— *新增内容*

"新增内容"链接会打开更新日志。您可以通过 X 按钮关闭横幅。

!!! note "注意"
    版本检查结果缓存 24 小时。重启服务器（或 Docker 容器）可以清除缓存并强制重新检查。
