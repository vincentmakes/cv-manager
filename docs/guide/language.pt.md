# Idioma

Clique no **ícone de globo** na barra de ferramentas para alterar o idioma da interface admin. Uma grade suspensa mostra todos os idiomas disponíveis — clique em um para aplicá-lo imediatamente.

**Idiomas suportados:** Inglês, Alemão (Deutsch), Francês (Français), Holandês (Nederlands), Espanhol (Español), Italiano (Italiano), Português (Português), Chinês (中文).

A configuração de idioma afeta apenas a interface admin — ela não altera o conteúdo do seu CV. Sua preferência é salva e persiste entre sessões.
