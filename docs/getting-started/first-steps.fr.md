# Premiers pas

Après avoir installé CV Manager, suivez ces étapes pour configurer votre CV :

1. Ouvrez l'interface d'administration à l'adresse `http://votre-serveur:3000`
2. Cliquez sur l'**icône de modification** dans l'en-tête pour définir votre nom, titre, biographie et coordonnées
3. Ajoutez vos expériences professionnelles, formations, certifications, compétences et projets
4. Utilisez le bouton **Thème** dans la barre d'outils pour choisir une couleur qui correspond à votre style
5. Cliquez sur **Imprimer / PDF** pour exporter, ou partagez l'URL publique sur le port 3001

!!! tip "Conseil"
    Le site public sur le port 3001 affiche une version en lecture seule de votre CV. Partagez cette URL avec les recruteurs et les employeurs — ils ne verront aucun contrôle d'administration.
