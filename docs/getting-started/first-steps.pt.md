# Primeiras Configurações

Após instalar o CV Manager, siga estes passos para configurar seu CV:

1. Abra a interface admin em `http://seu-servidor:3000`
2. Clique no **ícone de edição** no cabeçalho para definir seu nome, título, bio e informações de contato
3. Adicione suas experiências profissionais, educação, certificações, habilidades e projetos
4. Use o botão **Tema** na barra de ferramentas para escolher uma cor que combine com seu estilo
5. Clique em **Imprimir / PDF** para exportar, ou compartilhe a URL pública na porta 3001

!!! tip "Dica"
    O site público na porta 3001 exibe uma versão somente leitura do seu CV. Compartilhe essa URL com recrutadores e empregadores — eles não verão nenhum controle de administração.
