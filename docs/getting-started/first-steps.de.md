# Erste Schritte

Befolgen Sie nach der Installation von CV Manager diese Schritte, um Ihren Lebenslauf einzurichten:

1. Öffnen Sie die Admin-Oberfläche unter `http://ihr-server:3000`
2. Klicken Sie auf das **Bearbeiten-Symbol** in der Kopfzeile, um Ihren Namen, Titel, Bio und Kontaktdaten einzugeben
3. Fügen Sie Ihre Berufserfahrungen, Ausbildung, Zertifizierungen, Fähigkeiten und Projekte hinzu
4. Verwenden Sie die Schaltfläche **Design** in der Werkzeugleiste, um eine passende Farbe auszuwählen
5. Klicken Sie auf **Drucken / PDF**, um zu exportieren, oder teilen Sie die öffentliche URL auf Port 3001

!!! tip "Tipp"
    Die öffentliche Seite auf Port 3001 zeigt eine schreibgeschützte Version Ihres Lebenslaufs. Teilen Sie diese URL mit Personalvermittlern und Arbeitgebern — sie sehen keine Admin-Steuerelemente.
