# 快速开始

CV Manager 是一个通过 Docker 运行在您自己服务器上的 Web 应用程序。它提供两个界面：

- **管理端**（默认端口 3000）— 用于构建和管理您的简历
- **公开端**（默认端口 3001）— 只读版本，可分享给招聘人员、雇主或任何人

您的数据以 SQLite 数据库的形式存储在本地。不会向外部服务器发送任何数据。
