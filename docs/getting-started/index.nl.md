# Aan de slag

CV Manager is een webapplicatie die op uw eigen server draait (via Docker). Het biedt u twee interfaces:

- **Admin** (standaardpoort 3000) — waar u uw CV opbouwt en beheert
- **Publiek** (standaardpoort 3001) — een alleen-lezen versie die u kunt delen met recruiters, werkgevers of anderen

Uw gegevens worden lokaal opgeslagen in een SQLite-database. Er wordt niets naar externe servers verstuurd.
