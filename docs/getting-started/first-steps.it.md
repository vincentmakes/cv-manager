# Primi passi

Dopo aver installato CV Manager, seguite questi passaggi per configurare il vostro CV:

1. Aprite l'interfaccia admin all'indirizzo `http://your-server:3000`
2. Fate clic sull'**icona di modifica** nell'intestazione per impostare nome, titolo, biografia e informazioni di contatto
3. Aggiungete le vostre esperienze lavorative, formazione, certificazioni, competenze e progetti
4. Utilizzate il pulsante **Tema** nella barra degli strumenti per scegliere un colore adatto al vostro stile
5. Premete **Stampa / PDF** per esportare, oppure condividete l'URL pubblico sulla porta 3001

!!! tip "Suggerimento"
    Il sito pubblico sulla porta 3001 serve una versione di sola lettura del CV. Condividete quell'URL con recruiter e datori di lavoro — non vedranno alcun controllo di amministrazione.
