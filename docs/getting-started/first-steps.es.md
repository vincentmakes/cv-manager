# Primeros pasos

Después de instalar CV Manager, siga estos pasos para configurar su CV:

1. Abra la interfaz de administración en `http://your-server:3000`
2. Haga clic en el **ícono de edición** en el encabezado para establecer su nombre, título, biografía e información de contacto
3. Agregue sus experiencias laborales, educación, certificaciones, habilidades y proyectos
4. Use el botón **Tema** en la barra de herramientas para elegir un color que se adapte a su estilo
5. Presione **Imprimir / PDF** para exportar, o comparta la URL pública en el puerto 3001

!!! tip "Consejo"
    El sitio público en el puerto 3001 sirve una versión de solo lectura de su CV. Comparta esa URL con reclutadores y empleadores — no verán ningún control de administración.
